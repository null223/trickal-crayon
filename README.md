# Trickal クレヨン管理ツール

トリッカル・もちもちほっペ大作戦のクレヨン(バフ)管理ツール

## 機能

- ✅ ボード別・バフ種類別に使徒を整理
- ✅ チェックボックスで取得状況を管理
- ✅ ローカルストレージで永続化
- ✅ 検索・フィルター機能
- ✅ 進捗表示
- ✅ 使徒の画像表示（プレースホルダー自動生成）

## 使い方

1. `index.html` をブラウザで開く
2. 各使徒のチェックボックスをクリックして取得状況を管理
3. 検索ボックスで使徒名を検索
4. ボードフィルターで特定のボードのみ表示

## 画像について

現在、使徒の画像はプレースホルダー（カラフルな正方形に使徒名の最初の文字）が表示されています。

### 実際の画像を使用する場合

1. `images/characters/` ディレクトリに使徒の画像を配置
2. ファイル名は `[使徒名].png` の形式（例: `アリス.png`）
3. 推奨サイズ: 60x60ピクセル以上

画像が見つからない場合は、自動的にプレースホルダーが表示されます。

## ファイル構成

```
trickal/
├── index.html          # メインHTML
├── style.css           # スタイルシート
├── script.js           # JavaScript（データ・ロジック）
├── images/
│   └── characters/     # 使徒の画像ディレクトリ
└── README.md           # このファイル
```

- HTML5 + CSS3 + Vanilla JavaScript
- ローカルストレージで状態管理
- レスポンシブデザイン
- Canvas APIでプレースホルダー画像生成
